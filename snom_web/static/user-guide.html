User Guide

s-SNOM Spectrum Calculator calculates near-field optical spectra using the spheroid model for scattering-type scanning near-field optical microscopy.

Back to calculator

Basic Workflow

  1. Select Bulk sample or Layered sample.
  2. Enter tip geometry, tip permittivity, reference material, and harmonic number.
  3. Choose the frequency range, algorithm, and discretization parameters.
  4. Choose built-in materials, upload permittivity files, or define Drude-Lorentz models.
  5. Optionally upload an experimental spectrum.
  6. Press Calculate, inspect plots, and save CSV results if needed.

Models

Bulk Sample

Use this mode for a homogeneous semi-infinite sample. For a uniaxial sample with optical axis normal to the surface, use the effective permittivity sqrt(epsilon_parallel * epsilon_perpendicular).

Layered Sample

Use this mode for thin isotropic layers on a semi-infinite substrate. Available geometries are 1 layer on substrate and 2 layers on substrate.

Layer order is always: Tip, Layer 1, optional Layer 2, Substrate.

If far-field contribution is enabled, the calculator uses the indirect illumination coefficient ς and illumination angle θ.

Materials

Bulk built-ins: PMMA, SiO2, 4H-SiC, 6H-SiC.

Layer built-ins: PMMA, SiO2, 3C-SiC.

Substrate built-ins: Si, SiO2, 3C-SiC.

Uploaded permittivity files must contain at least three numeric columns:

frequency_cm-1 Re_epsilon Im_epsilon

The selected frequency range must be inside the uploaded file range.

Drude-Lorentz Model

The exact convention used by the calculator is:

ε(ω) = ε∞ - ωp² / (ω² + iγDω) + ∑ fjωj² / (ωj² - ω² - iγjω)

Use Drude only, Lorentz only, or both. At least one Drude term or one Lorentz oscillator is required.

Algorithms

Fast uses fixed N = 30 and M = 30 for quick calculations.

Accurate allows custom N and M. Limits are N <= 500 and M <= 100.

Layered calculations also use the number of image charges K. The implemented limit is K <= 50.

Experimental Spectrum

Experimental files must contain:

frequency_cm-1 amplitude phase

The experimental spectrum is overlaid on amplitude and phase plots. If present, the calculator also shows amplitude and phase residuals, simulation minus experiment, over the overlapping frequency range.

Plots, History, and Saving

The calculator shows near-field amplitude |σn|, phase arg(σn), and material permittivity. Layered mode plots all layer and substrate permittivities.

The last 10 calculated spectra are kept in browser memory for the current session. The newest spectrum is visible by default. Older spectra remain in the legend hidden by default and can be shown by clicking their legend entries.

Save the results downloads a CSV for the latest calculation, including all relevant permittivity columns, Re/Im σn, |σn|, and arg(σn).
